# Budakattu Sante – Forest-to-Home Marketplace

## 📌 Project Overview
Budakattu Sante is an Android application developed to support tribal artisans and rural communities by providing a direct marketplace between sellers and customers. The application helps eliminate middlemen exploitation and promotes fair trade for tribal products such as wild honey, bamboo crafts, handmade items, and organic forest products.

The project focuses on improving rural livelihoods through digital connectivity and smart marketplace solutions.

---

## 🎯 Objectives
- Provide direct market access for tribal artisans
- Promote fair pricing and transparent trade
- Digitize rural product sales
- Support local and eco-friendly products
- Create a simple and user-friendly mobile marketplace

---

## ✨ Features
- User Login & Registration
- Product Listing Interface
- Product Categories
- Search & Navigation
- Cart Management
- Order Placement
- Local Database Support using Room DB
- Smooth Navigation using Navigation Components
- Modern Android UI with ViewBinding

---
